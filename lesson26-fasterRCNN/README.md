# Faster RCNN

Faster R-CNN R-101-FPN model was implemented with tensorflow 2.0.

# Requirements

- python 3.*
- tensorflow 2.*
- cv2

# Usage

```
python train_model.py


```
see `train_model.ipynb` and `inspect_model.ipynb` for some insight.
